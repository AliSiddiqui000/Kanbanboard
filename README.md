# Kanban Board Application

## Project Description

The Kanban Board Application is a project management tool designed to help users organize tasks visually. It allows users to create, edit, and manage tasks across different statuses (To Do, In Progress, Done) using a simple and intuitive interface. The application supports comments on tasks, image attachments, and integrates with Google Drive for file management.

## Features
- **Task Management**: Create, edit, and delete tasks.
- **Visual Organization**: Drag and drop tasks across different status columns.
- **Comments**: Add comments to tasks for better collaboration.
- **Image Attachments**: Attach images to tasks for additional context.
- **Google Drive Integration**: Upload and manage files directly from Google Drive.
- **Notifications**: Vibration alerts for tasks marked as "In Progress" after 15 minutes.
- **Dynamic App Icon**: Change the app icon when the app goes to the background on iOS.

## Purpose

This project aims to provide a flexible and user-friendly platform for managing tasks and collaborating on projects. It is built using React Native, enabling cross-platform functionality for both iOS and Android devices. The Kanban board methodology allows users to visualize their workflow and manage tasks efficiently.

## Setup Instructions

### Prerequisites

Before running the project, ensure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) (LTS version recommended)
- [Expo CLI](https://docs.expo.dev/get-started/installation/) (for development and testing)
- An IDE or text editor (e.g., Visual Studio Code)

To run the project on an emulator, use the following commands:
